FolderRow — a Relvo memory « dossier » as a colour-coded list row (Mémoire screen).

```jsx
<FolderRow name="Fournisseurs" sub="8 sujets · 12 documents" color="var(--blue-600)" icon={<Package/>} onClick={open} />
<FolderRow name="RH" sub="3 sujets · 5 documents" color="var(--purple-600)" icon={<Users/>} />
```

Domain colours match the SubjectRow rails: Fournisseurs blue, RH purple, Juridique amber, Clients accent-red, Production green.
