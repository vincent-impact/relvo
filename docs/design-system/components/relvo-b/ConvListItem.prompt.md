ConvListItem — one row in the « Mes conversations » list (what the header logo opens).

```jsx
<ConvListItem title="Réponse à Karim — sauce blanche"
  preview="Vous pourrez lui proposer la substitution…" when="5 min" context="SUB-0142" onClick={open} />
```

Show the `context` chip when the conversation is tied to a Subject. Put a highlighted "Nouvelle conversation" entry above the list.
